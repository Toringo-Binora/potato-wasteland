# GitHub Pages Checklist

1. Push this repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/ (root)`.
6. Save and wait for the site to publish.
7. Open `https://YOUR-USERNAME.github.io/YOUR-REPO/`.

## Included Pages Prep

- The game now loads Three.js from `vendor/three.module.min.js`.
- `node_modules/` is ignored and does not need to be pushed.
- The site is fully static, so no build step is required.
